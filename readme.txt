The Mobile Device Redirection plugin allows your Wordpress site to redirect to another url if a user visits it using a mobile device.

== Description ==

Mobile Device Redirection, using detection from the MobileESP project (http://www.mobileesp.com).

See the Frequently Asked Questions for guidance on how to use the plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `mobile-device-redirection` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings->Mobile Device Redirection and choose the url you what users redirected to when a mobile device encounters your page.
